# Fitness Social Platform (FitPlanHub) https://fitplanhuben22cs301050.pages.dev/

A social media platform for fitness enthusiasts and trainers to connect and share workout plans.

## Features

- **User Authentication**
  - Sign up and login for both users and trainers
  - Secure password protection
  - Profile management

- **Social Feed**
  - View posts from all trainers
  - Like and comment on posts
  - Follow/unfollow trainers
  - View post details and engagement

- **Trainer Dashboard**
  - Create and manage fitness plans
  - Set pricing for premium content
  - Track followers and engagement
  - Update profile and plan details

- **Discover Section**
  - Browse all trainers and their profiles
  - Search and filter functionality
  - View trainer details and plans

- **Responsive Design**
  - Works on desktop and mobile devices
  - Dark theme support
  - Intuitive user interface

## Tech Stack

- Frontend: React with TypeScript
- Styling: Tailwind CSS
- Backend: Supabase
- Database: PostgreSQL

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/fitness-social-app.git
   cd fitness-social-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Copy `.env.example` to `.env`
   - Add your Supabase credentials

4. Start development server:
   ```bash
   npm run dev
   ```

5. Open in browser:
   ```
   http://localhost:5173
   ```

## Usage

- Sign up as a user or trainer
- Browse and follow trainers
- Like and comment on posts
- Trainers can create and manage workout plans
